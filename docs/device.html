run_frida_server(frida_server_version='14.2.18')[source]¶

Run frida server of the given version on the device. First it will check if the binary is already exists on the device. If so, will run it. If it doesn’t exist, it will check if the binary exists in the package’s store. If so, it will take it from there. Otherwise, it will try to download the binary from the internet, upload it and run it on the device.

Parameters

frida_server_version (str) – the frida server’s version to run on the device

Raises

Exception – If the device is not rooted or if failed to run frida server on the device.
